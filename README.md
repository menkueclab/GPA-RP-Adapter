# GPA-RP-Adapter
This is a shield for the red pitaya (rp-122 or rp-125) to connect it easily to the GPA-FHDO.

# Parts
- pin header
- RJ45 connector
- schmitt trigger (optional)
- SMA connector  (optional)
- capacitor  (optional)
- fan (optional)

for details see KiCad schematic.
